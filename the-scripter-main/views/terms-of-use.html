Terms of Usage

Cloud Based Code-Execution Platform facilitates programmers to execute their multiple programs and scripts in cloud servers simultaneously. Each programmer may use a different problem solving environment. Hence our approach is to provide a secure platform that supports multiple problem solving environments through which programmers can execute code parallelly in a cloud environment, even if the required compiler/interpreter is not installed locally by the programmer. Through our Code-Execution platform programmers can upload and execute their programs for multiple days or even months depending upon their need. We offer background execution of your programs in our cloud servers for up to two months.

Usage of modules like pyautogui, tkinter, os, plyer are not allowed because these modules simulates chnaages in server. As your code is executing on cloud server you cannot expect the output and changes of these modules in your personal computer.

Once you click on RUN button your code will be started execution automatically in cloud servers. The best way to make use of this platform is by executing web scraping programs.

Finally, you can make use of modules like Twilio for makeing phone calls using python. For example if your condition is getting satisifed then you can place a phone call to your personal mobile number.

For example, let us consider you have written a program which will send an alert to your mobile number when the stock price of XYZ company reaches 4,000/- Now no one know how many days it will take for stock price of XYZ company to reach 4,000/- Hence you cannot execute these kind of programs in your personal computer because personal computers cannot run for multiple days.

So, we are introducing our platform where you can run these kind of programs and finally you will get your desired output when the condition in your code is getting satisfied.
